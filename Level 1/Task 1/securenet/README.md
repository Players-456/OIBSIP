## **SecureNet Landing Page** 

This project is a responsive cybersecurity-themed landing page developed using HTML, CSS, and JavaScript as part of the Oasis Infobyte Web Development Internship. 

The webpage focuses on creating a modern and visually appealing interface for a fictional cybersecurity service platform called SecureNet. 

## **Features** 

- Responsive landing page design 

- Modern cybersecurity-themed UI 

- Navigation bar with smooth layout 

- Hero section with call-to-action 

- Services and information sections 

- Clean animations and styling 

- Mobile-friendly interface 

## **Technologies Used** 

- HTML5 

- CSS3 

- JavaScript

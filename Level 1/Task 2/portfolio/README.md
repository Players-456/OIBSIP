## **Personal Portfolio Website** 

This project is a responsive Personal Portfolio Website developed using HTML, CSS, and JavaScript as part of the Oasis Infobyte Web Development Internship. 

The portfolio showcases personal information, skills, projects, and contact details in a professional and organized layout. 

## **Features** 

- Responsive portfolio design 

- About section 

- Skills showcase 

- Projects section 

- Contact information section 

- Smooth scrolling navigation 

- Clean and modern UI 

## **Technologies Used** 

- HTML5 

- CSS3 

- JavaScript

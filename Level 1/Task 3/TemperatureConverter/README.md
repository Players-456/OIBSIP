## **Temperature Converter** 

This project is a Temperature Converter web application developed using HTML, CSS, and JavaScript as part of the Oasis Infobyte Web Development Internship. 

The application allows users to convert temperature values between Celsius, Fahrenheit, and Kelvin instantly. 

## **Features** 

- Celsius to Fahrenheit conversion 

- Fahrenheit to Celsius conversion 

- Kelvin conversions 

- Instant calculation results 

- User-friendly interface 

- Responsive design 

## **Technologies Used** 

- HTML5 

- CSS3 

- JavaScript

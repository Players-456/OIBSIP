## **Oasis Infobyte Web Development Internship —** 

## **Level 3** 

This repository contains the Level 3 task completed during the Oasis Infobyte Web Development Internship. The project focuses on full-stack web development using the MERN stack along with authentication, order management, payment integration, and admin dashboard functionality. 

The objective of Level 3 was to build a real-world scalable web application with modern frontend architecture, backend APIs, authentication systems, and database management. 

## **Project Overview** 

## **Task 1 — Pizza Delivery MERN Application** 

A full-stack pizza delivery web application built using the MERN stack. The platform allows users to browse pizzas, customize orders, manage carts, place orders, and access admin management features. 

The project includes authentication, inventory management, payment integration, custom pizza building, and role-based admin functionality.

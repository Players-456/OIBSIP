## **To-Do Application** 

This project is a responsive To-Do Application developed using HTML, CSS, and JavaScript as part of the Oasis Infobyte Web Development Internship. 

The application allows users to create, manage, update, and delete daily tasks in a simple and organized way. 

## **Features** 

- Add new tasks 

- Mark tasks as completed 

- Delete tasks 

- Dynamic task management 

- Responsive user interface 

- Clean and minimal design 

## **Technologies Used** 

- HTML5 

- CSS3 

- JavaScript

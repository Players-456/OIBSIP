## **Tribute Page** 

This project is a responsive Tribute Page developed using HTML, CSS, and JavaScript as part of the Oasis Infobyte Web Development Internship. 

The webpage is designed to present information, achievements, and important details about a well-known personality in a visually appealing format. 

## **Features** 

- Responsive webpage design 

- Structured content sections 

- Image and biography section 

- Clean typography and layout 

- Simple and readable UI 

- Smooth scrolling interface 

## **Technologies Used** 

- HTML5 

- CSS3 

- JavaScript

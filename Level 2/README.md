## **Oasis Infobyte Web Development Internship —** 

## **Level 2** 

This repository contains all Level 2 tasks completed during the Oasis Infobyte Web Development Internship. These projects focus on frontend interactivity, JavaScript application development, and authenticationbased full-stack systems. 

The objective of Level 2 was to strengthen frontend and backend development skills by building practical web applications and real-world authentication systems.

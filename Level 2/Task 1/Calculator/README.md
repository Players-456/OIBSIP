# Premium Calculator

Modern dark-theme calculator web app built for the **Oasis Infobyte Web Development Internship** (Level 2 — Task 1).

## Features
- Addition, Subtraction, Multiplication, Division
- Decimal support & Percentage
- Clear (AC), Backspace/Delete
- Chained calculations
- Divide by zero & invalid input error handling
- Copy result to clipboard
- Keyboard input support
- Calculation history (localStorage persisted)
- Click history item to reuse result
- Scientific mode: sin, cos, tan, √, log, ln, xʸ, π

## Keyboard Shortcuts
| Key | Action |
|-----|--------|
| `0–9`, `.` | Input |
| `+` `-` `*` `/` | Operators |
| `Enter` or `=` | Equals |
| `Backspace` | Delete |
| `Esc` or `c` | Clear |
| `%` | Percent |
